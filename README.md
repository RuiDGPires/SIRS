## Frontend

### Setup

`sudo apt update`
`sudo apt install npm`
`npm install`

You may need: `npm config set strict-ssl false`

#### Unexpected token on npm start (Possible fix)

`curl https://raw.githubusercontent.com/creationix/nvm/v0.30.2/install.sh | bash`

`nvm install 18.0.0`

Try again

### Run

`npm start`
